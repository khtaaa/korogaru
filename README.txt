作品名　YOKERO
作成者　加幡　祐人


・起動方法　buildファイルのYOKERO.exeを起動してください
・プログラム　Assetsファイルのscriptファイルに入っているのでMonoDevelop、Visual Studioなどで起動してください
・BGM、SE　BGMやSEは魔王魂さんの音を使用させていただいています。http://maoudamashii.jokersounds.com/

このミニゲームは何かゲームを作ろうと思ってたときにぱっと思い浮かんで作ったものです。
このゲームで一番苦労したことは生成したボールに方向などを加えるところでした、
まだギミックは多いとはいえないので今後ギミックを増やして生きたいと思います
